memcache基础

Posted on 2020-04-07 In memcache

  1. set() 增/改
    语法: set key msg cache_time length
    key: 不能超过250字符
    msg: key 的额外信息 (整型参数)
    cache_time(/s): 0-理论永久 (最大30天)
    length: 数字类型,单位字节

  2. get()
    语法: get key

  3. replace()
    语法: replace key 是否压缩(0/1) cache_time/s length/字节
    只能修改不能添加

  4. add()
    语法: add key 是否压缩 cache_time/s length/字节
    只能添加不能修改

  5. delete()
    语法: delete key

  6. incr() decr()
    语法: ``` incr key nums 

    decr key nums  ```

    返回增长后的接口 ,key必须存在

  7. flush_all

set() add() 区别

add添加key时如果存在会返回false  
set会直接覆盖 相当于add 和replace的载体
